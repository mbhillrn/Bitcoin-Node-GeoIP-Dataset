# Bitcoin Node GeoIP Dataset

Maintained database of geolocated IP (ipv4, ipv6) addresses used For the MBCore Dashboard. 
Pre-cached geolocation data for known Bitcoin Core peer IP addresses (Lat, lon, city, region/state, countery, continent, isp, etc)

Used by [MBCore Dashboard](https://github.com/mbhillrn/Bitcoin-Core-Peer-Map) to provide instant location lookups without API calls.

## Updates

This database may or may not be updated periodically.

## Data Source

- Data found online through multiple sources using bitcoin-cli tools and geolocation apis (free-tier / non-commercial, such as ip-api.com)
