# Bitcoin Node GeoIP Dataset

Pre-cached geolocation data for known Bitcoin Core peer IP addresses.

Used by [MBCore Dashboard](https://github.com/mbhillrn/Bitcoin-Core-Peer-Map) to provide instant location lookups without API calls.

## Data Source
- Geolocation: ip-api.com (free tier, non-commercial)
- IPs: Collected from Bitcoin Core `getpeerinfo`
